# Pattern-Memory-Game

A brain testing game where you have to remember given elements in order and choose the same from given options. The pattern only appears for a few seconds, so players need quick focus and skills to win each round.

## Features 

-I have tried my best to give the game an aesthetic look 
-You get 8s to memorize a sequence, then 20s to rebuild it in same order  
-A smooth and... cool live animated time bar that changes color with time running (Hope it actually looks cool) 
-Live Score System 
-3 Lives provided in total, with visual life indicator
-Resign Feautre

## Gameplay

At the start of each round, a random sequence is shown on screen for a limited amount of time. 
Once the sequence disappears, player must click the correct options in the same order as shown earlier. 
If a wrong option is selected or the timer reaches zero one life is lost. 
The game continues until all lives are gone.
Every successful round increases score and generates a new random patterns.
